Node.jsâ€‹

You need to have Node.js installed. Any stable (LTS >= 14.0) version is sufficient.

Dependenciesâ€‹

Navigate into your newly created folder kilt-rocks, initialize the project and install dependencies.

  • KILT SDK-JS - for KILT functionality
  • dotenv - to help us load variables from .env
  • ts-node and Typescript - to execute our TS code
  • npm
  • Yarn
npm init -y
npm install @kiltprotocol/sdk-js@0.28.1 dotenv ts-node typescript
yarn init -y
yarn add @kiltprotocol/sdk-js@0.28.1 dotenv ts-node typescript

PILT Tokensâ€‹

In this workshop, you will interact with the Peregrine blockchain. You are required to pay for each transaction with PILT coins. But worry not! PILT coins don't have any value and can be requested from the faucet.

Blockchain Connectionâ€‹

Before you call any SDK functionality, you need to initialize the crypto libraries and configure the SDK. For this workshop, we'll be using Peregrine Testnet. This is done by calling await Kilt.init({ address }) where address is the address of the full node you want to connect to. For this workshop, use wss://peregrine.kilt.io/parachain-public-ws. Add the address to your .env file.

.env
WSS_ADDRESS=wss://peregrine.kilt.io/parachain-public-ws

That's it for the basic setup - We're good to go!
